# 문자 반복 출력하기
## **[문제]**
문자열 **my_string**과 정수 **n**이 매개변수로 주어질 때,<br>
**my_string**에 들어있는 각 문자를 **n**만큼 반복한 문자열을 return 하도록 solution 함수를 완성해주세요.<br>
<br>

## **[제한사항]**
* 2 <= my_string 길이 <= 5
* 2 <= n <= 10
* my_string은 영어 대소문자로 이루어져 있습니다.
<br><br>

## **[입출력 예]**
ex.1) my_string="hello" / n=3<br>
