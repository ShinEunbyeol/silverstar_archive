# 짝수의 합
## **[문제]**
정수 **n**이 주어질 때,<br>
**n**이하의 짝수를 모두 더한 값을 return 하도록 solution 함수를 작성해주세요.<br>
<br>

## **[제한사항]**
* 0 < n <= 1,000
<br><br>

## **[입출력 예]**
ex.1) n=10<br>
&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;result=30<br>
ex.2) n=4<br>
&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;result=6<br>
